📊 Superstore Sales Performance Dashboard
Project Overview
This project is a comprehensive sales analysis of a Superstore dataset using Microsoft Excel. It helps business owners track revenue, profitability, and customer trends across different regions and categories.

Key Features
Interactive Dashboard: Built using Pivot Charts and Slicers.

Dynamic KPIs: Real-time tracking of Total Sales and Profit.

Data Cleaning: Processed raw data to ensure accuracy.
